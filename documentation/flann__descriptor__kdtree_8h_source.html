40 // A wrapper around the Fast Library for Approximate Nearest Neighbor search.
41 // This is useful for matching descriptors without having to run all pairwise
42 // comparisons. The FLANN kd tree builds up a high-dimensional spatial index of
43 // all input descriptors, allowing for fast but approximate nearest-neighbor
44 // queries in log(n) time, where n is the number of dimensions in the
45 // descriptor.
46 //
